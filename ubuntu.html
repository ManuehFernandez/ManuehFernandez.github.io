Ubuntu

Historia de Ubuntu

Ubuntu es una distribución de Linux que incluye principalmente software libre y de código abierto. Es una bifurcación de Debian. El objetivo inicial era hacer de Debian una distribución más fácil de usar y entender para los usuarios finales, corrigiendo varios errores de este y haciendo más sencillas algunas tareas como la gestión de programas.
Puede utilizarse en ordenadores y servidores y está orientado al usuario promedio, con un fuerte enfoque en la facilidad de uso y en mejorar la experiencia del usuario. Está compuesto de múltiple software normalmente distribuido bajo una licencia libre o de código abierto.
Ubuntu fue fundado por Canonical, una compañía británica propiedad del empresario sudafricano Mark Shuttleworth. Ofrece el sistema de manera gratuita, y se financia por medio de servicios vinculados al sistema operativo y vendiendo soporte técnico.
Desde el inicio del proyecto, Shuttleworth proporcionó el soporte económico gracias a los beneficios obtenidos después de vender su empresa, Thawte, por unos 575 millones de dólares estadounidenses.
El 8 de julio de 2005, Shuttleworth anunció la creación de la Fundación Ubuntu con el propósito de asegurar soporte y desarrollo para todas las futuras versiones de Ubuntu. Aportaron 10 millones de dólares como presupuesto inicial.
Al mantenerlo libre y gratuito, la empresa es capaz de aprovechar los desarrolladores de la comunidad para mejorar los componentes de su sistema operativo. A de Debian, Canonical asegura una nueva versión de Ubuntu cada seis meses con soporte asegurado durante nueve meses. Estas versiones actualizadas incluyen actualizaciones de seguridad y de programas y parches para bugs.
Antes de cada lanzamiento, se lleva a cabo una importación de paquetes desde Debian a las que les aplican las modificaciones específicas de Ubuntu. Un mes antes del lanzamiento, comienza un proceso de congelación de importaciones, ayudando a que los desarrolladores puedan asegurar un software estable.
Extraoficialmente, la comunidad de desarrolladores proporciona soporte para otras derivaciones de Ubuntu, como Kubuntu, Xubuntu, Ubuntu MATE, Edubuntu, etc.
Canonical, también ofrece otras versiones de Ubuntu orientadas a distintos servicios: Ubuntu Server (una versión para servidores), Ubuntu Business Desktop Remix (una versión para empresas), Ubuntu TV, Ubuntu Tablet y Ubuntu Phone, entre muchas otras.
En 2009, Canonical empieza a tener problemas con la gestión de paquetes e instalación de aplicaciones. Es demasiado fragmentada y compleja. Así, se planifica la creación de una aplicación central para el manejo e instalación de aplicaciones: el Centro de software de Ubuntu, que permite buscar, instalar y desinstalar aplicaciones y agregar repositorios de terceros.

Características de Ubuntu

  • Ubuntu es versátil y puede instalarse en computadores de escritorio, laptops y cualquier tipo de dispositivo de telecomunicaciones, funciona adecuadamente y presta un servicio eficiente en todo momento, por lo que resulta muy solicitado por el público en general.
  • Incluye controladores Nvidia a través de los cuales, los fabricantes proporcionan el uso de tarjetas gráficas, con el gran beneficio de incrementar el rendimiento de todos los equipos donde se hace esta instalación y por otro lado aporta una gran reducción de tiempo, en el proceso de configuración del sistema luego de la instalación correspondiente.
  • Arranque y apagado veloz
  • Código abierto y libre distribución. Cualquier usuario o empresa puede desarrollar a Linux, agregando mayor calidad a las funciones y características del programa, se trata de un sistema operativo completamente gratis, que cultiva en forma positiva todos los aportes correspondientes, a través de una gran cantidad de usuarios a nivel mundial.
  • Una de las metas principales de Linux Ubuntu, es que está dirigida hacia el trabajo y comunicaciones en redes, con todo el beneficio que esto trae para el público en general.
  • Es compatible con la gran mayoría de hardware, aprovechando además que existe una enorme facilidad en los requisitos básicos para instalarlo.
  • Se facilita enormemente el funcionamiento con un alto nivel de seguridad dando así a sus suscriptores, una tranquilidad y funcionalidad adecuada de mucha confianza.
  • No está sujeto a ninguna multinacional, país u organización. Son los usuarios los que tienen la responsabilidad de actualizar, implementar y desarrollar el sistema de Linux y en ningún caso existe compromiso alguno con empresas multinacionales o de otro tipo.

Versiones de Ubuntu

Las versiones de Ubuntu tienen un nombre en clave bastante curioso. Éste está compuesto por el nombre de un animal y un adjetivo que empieza por la misma letra de dicho animal. Además, desde la versión 6.06 (Dapper Drake), estos nombres han seguido una progresión por orden alfabético. Cada versión, también incluye un fondo de escritorio con el animal del nombre que tenga la expansión.
